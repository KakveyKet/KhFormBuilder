# Contributing to KhFormBuilder

Thank you for considering contributing to **KhFormBuilder**! We welcome contributions from the community to make this project better for everyone.

## How to Contribute

### 1. Fork the Repository
To begin contributing, fork the repository by clicking the **Fork** button at the top-right of the repository page.

### 2. Clone the Repository
Once you have forked the repository, clone it to your local machine:
```bash
git clone https://github.com/KakveyKet/KhFormBuilder.git
cd KhFormBuilder

### 3. Set Up Your Local Environment
Make sure to set up the development environment by installing the required dependencies. In most cases, you can do this with:
```bash
npm install
